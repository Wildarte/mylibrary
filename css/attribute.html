Utilizando css attribute selector

Podemos usar o attribute selector do CSS da seguinte forma: tag[attribute="value"]
Exemplo:

            h2[title="titulo"]{
                color: red
            }
Dessa forma como exemplo acima mostra todo h2 que tiver um atributo title com escrito "titulo", ficará com a cor vermelha
